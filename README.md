# bbcent7
Vagrant BaseBox for CentOS 7l

- git clone https://github.com/ctrees/bbcent7.git
- cd bbcent7
- vagrant login (cat color#test)
- vagrant init cat/bbcent7
- vagrant up --provider virtualbox
